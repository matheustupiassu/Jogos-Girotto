# Jogos-Girotto

Integrantes do grupo: Rodrigo Martins do Nascimento e Matheus Tupiassu de Pinho Lourenço.

Foram utilizadas as bibliotecas: 

- stdio.h (entrada e saída básica de dados);

- time.h (semente de aleatoriedade)

- stdlib.h

- locale.h (para a detecção de caracteres especiais e a possibilidade de uso de acentos e ç);

- string.h (para conseguir manipular strings mais facilmente, sendo utilizado principalmente para se referir a uma string de maneira mais simples);

- windows.h (usada a função sleep para ter um temporizador melhor);

O define foi usado para definir e dar cor às linhas do código, deixando-o mais bonito.

O primeiro jogo é mais simples, sendo basicamente um sistema de if e else, com modificações para que caracteres especiais e letras sejam detectados como opções inválidas.

O segundo jogo usa a semente de aleatoriedade, manipulação de string para facilitar o uso dos nomes de personagens, arrays para organização e definição das caixas (funcionando como uma matriz, em que 1 representa um botão e 2 uma cobra).

O terceiro jogo usa um sistema um pouco mais complexo de matrizes para definir cada jogador e suas gousmas, além de loops de repetição para contagem e ações, fazendo com que o código seja um pouco mais curto do que um usando outra maneira de definição de gousma.

foi usado o Gemini (quando a solução para o problema não era encontrada), além de para aprender sobre como utilizar bibliotecas não explicadas em sala ou outras funções interessantes possíveis de se fazer em C para um código mais eficiente e dinâmico.
